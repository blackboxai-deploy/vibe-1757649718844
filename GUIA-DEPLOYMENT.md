## 📖 Instrucciones Paso a Paso

### **OPCIÓN 1: GitHub Pages (Más fácil)**

1. **Crear repositorio:**
   ```
   - Ve a github.com
   - Clic en "New repository"
   - Nombre: "calculadora-depreciacion" 
   - Marca "Public"
   - Clic "Create repository"
   ```

2. **Subir archivo:**
   ```
   - Clic "uploading an existing file"
   - Arrastra "calculadora-depreciacion.html"
   - Renombra a "index.html"
   - Clic "Commit changes"
   ```

3. **Activar GitHub Pages:**
   ```
   - Ve a Settings del repositorio
   - Scroll hasta "Pages"
   - Source: "Deploy from a branch"
   - Branch: "main"
   - Folder: "/ (root)"
   - Clic "Save"
   ```

4. **¡Listo! Tu URL será:**
   ```
   https://tuusuario.github.io/calculadora-depreciacion
   ```

### **OPCIÓN 2: Netlify (Más moderno)**

1. **Subir directamente:**
   ```
   - Ve a netlify.com
   - Clic "Add new site" > "Deploy manually"
   - Arrastra tu archivo HTML
   - ¡Automáticamente se despliega!
   ```

2. **URL personalizada (opcional):**
   ```
   - Site settings > Change site name
   - Elige: "calculadora-depreciacion-tuapellido"
   - Tu URL: calculadora-depreciacion-tuapellido.netlify.app
   ```
